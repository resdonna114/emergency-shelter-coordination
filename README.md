# Emergency Shelter Coordination

A blockchain-based disaster response platform for emergency management and shelter coordination.

## Features

• **Shelter Capacity Management**: Register and track emergency shelter availability and occupancy
• **Resource Allocation**: Monitor and update shelter resources (food, water, medical supplies, blankets)
• **Evacuee Tracking**: Register evacuees and assign them to appropriate shelters
• **Volunteer Coordination**: Manage volunteer registration and skill tracking

Built with Clarity smart contracts on the Stacks blockchain for transparent and reliable emergency response coordination.
